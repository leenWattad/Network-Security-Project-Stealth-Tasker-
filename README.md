# Network-Security-Project-Stealth-Tasker-
This project involves developing a weekly task planner application with 
embedded Remote Access Trojan (RAT) capabilities, making it both a user
friendly productivity tool and an exploration of cybersecurity risks. 
The graphical interface, built with Tkinter, allows users to manage their weekly 
schedules, while the background processes silently establish a connection with a 
remote server, enabling full remote access to the host system.
